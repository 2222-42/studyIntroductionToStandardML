`- fun id x = x`の型は：

`val id = fn : 'a -> 'a`であり、これにより

`id = fn : ('a -> 'a) -> ('a -> 'a)`
-> `id id = fn : 'a -> 'a`となる。

次に、`val f = fn : 'a -> 'b`とする。

このとき`x: 'a`であるなら、`f x: 'b`となる。

よって、`- fn x => fx`の型は`'a -> 'b`となる。

`- fn x => id id x`の型は、よって、

最初の`id`の型は` ('a -> 'a) -> ('a -> 'a)`となり、
次の`id`の型は`'a -> 'a`となり、
`id id`の型は、`'a -> 'a`となり、よって、

`val it = fn : 'a -> 'a`となる。
